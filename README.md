# Responsive Landing Page with Tailwind CSS

A modern and fully responsive landing page built using Tailwind CSS. This project was created to practice responsive web design, utility-first styling, and modern UI development techniques.

## 🚀 Live Demo

https://akelvino.github.io/landigpage-using-tailwind/

## ✨ Features

* Responsive design for mobile, tablet, and desktop devices
* Modern and clean user interface
* Hero section with call-to-action buttons
* Features and services showcase
* Customer reviews/testimonials section
* Smooth navigation experience
* Tailwind CSS utility-first styling
* Fast-loading static website

## 🛠️ Technologies Used

* HTML5
* Tailwind CSS
* JavaScript
* Remix Icons

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/akelvino/landigpage-using-tailwind.git
```

Navigate to the project directory:

```bash
cd landigpage-using-tailwind
```

Open `index.html` in your browser or run a local development server.
